Wowee requires assets extracted from a compatible World of Warcraft client.

1. Drag Wowee.app to the Applications shortcut.
2. Double-click "Wowee Asset Extractor.app" from this disk image.
3. Select the World of Warcraft Data folder that contains the MPQ files.
4. Leave the Terminal window open until extraction completes.
5. Open Wowee.app.

Extracted files are stored in:
~/Library/Application Support/Wowee/Data

They remain available when Wowee.app is upgraded and do not modify the signed
application bundle.
